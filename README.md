# HMIS - Hospital Management Information System

A full-stack HMIS project using:

- Spring Boot (Backend)
- React + Vite (Frontend)
- PostgreSQL (Database)

## Project Structure

- `/backend/backend` -> Spring Boot API
- `/frontend` -> React single-page application

## How To Run

- backend -> `cd backend/backend;mvn clean spring-boot run`
- frontend -> `cd frontend; npm run dev`
